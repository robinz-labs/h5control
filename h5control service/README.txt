How to install and run h5control service ?
------------------------------------------

Mac OS X:
Copy app file package "h5control Service" to your disk and run it.

Windows:
Copy folder "h5control Service" to your disk and run "h5connect.exe" or "h5connect-console.exe" in the folder.



How to rebuild h5control service with the source code ?
-------------------------------------------------------

1, Install Qt SDK 5.7 or higher version in your operation system (Linux, Windows or Mac OS X).
2, Find the source code and open file H5Connect.pro in QtCreator.

What's h5control ?

h5control provides an easy way to enable your web app to control your own hardware with digital/analog IO, motors, servos, LED lights, sensors and other electronic gadgets.

  • Employs popular open-source hardware Arduino.
  • Easy and fast programming only uses HTML5/JavaScript for all.
  • Easy debugging with Web Inspector is same as the way for a common web app.
  • Your web app will connect your hardware via USB or Bluetooth.

 

Before your programming

  • Get an Arduino that is ready for h5control ( info ).
  • Install and run h5control service on your computer ( info ).
  • Connect the Arduino to your computer through USB.

 

Make your first h5control app

Create a web page file contains HTML code as below, and copy file h5control.js to the same directory of your html file.
Open it in your web browser ( Chrome is recommended ).

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
   <script type="text/javascript" src="h5control.js"></script>
   <script type="text/javascript">

      var led = new DigitalOut();

      window.onload = function() {
         h5control.setup();
      };

      h5control.whenConnect = function() {
         h5control.resetAll();
      };

      h5control.whenUnitReady = function(unitId) {
         led.setup(unitId, 13);
      };

      led.whenGetValue = function(value) {
         alert("LED is " + (value ? "ON" : "OFF"));
      };
   </script>
</head>

<body>
<p>
   <input type="button" value="led on" onclick="led.setValue(1);" />
   <input type="button" value="led off" onclick="led.setValue(0);" />
   <input type="button" value="led state ?" onclick="led.getValue();" />
</p>
</body>
</html>

Now, please open the web page in your web browser.
